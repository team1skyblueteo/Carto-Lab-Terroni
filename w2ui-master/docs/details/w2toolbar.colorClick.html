Called when user selects an item from a color picker drop down menu.
colorClick(event)
event object, event object
Returns undefined.

Description

This method only works for toolbar item of type color or text-color. It is called when user selects a color. It is used internally, but you can use to emulate clicks. This method emits .onClick event.
